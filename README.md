# Magic_number

01. magicnumber 
----------------
program to get magic number from file header

02.compile the program: 
----------------
$ gcc -o magic magicnumber.c

03.run program: 
----------------
$ ./magic ./test.txt

04. output example:
---------------
$ ./magic ./magic           
7f454c46
The magic number of your file is:
7f454c46c842
Size of file:
17096
